## Notation
* Σ represents an alphabet
* L = language
* M = DFA
* L(M) describes language accepted by machine
* L = {w| predicate over w} ∈ Σ*
* Σ* zero or more elements of the set
* Σ<sup>+</sup> one or more elements of the set
